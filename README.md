# 基于Django的线上自习室预约管理系统简介

## 核心功能
- 用户认证
- 自习室浏览
- 座位实时状态查看
- 在线预约
- 扫码签到
- 后台管理

## 适用场景
- 高校学生自习
- 社会人士学习
- 公共自习室管理

## 技术特点
- 采用Django框架，保障系统稳定性和安全性
- 前后端分离架构，提升用户交互体验
- 实时数据同步，确保座位状态准确无误
